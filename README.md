📖 Description

The Healthcare Expense Tracker + AI Insurance Advisor is a lightweight yet powerful web application built using Python and Flask. It enables users to efficiently record and manage their medical expenses, including costs related to medicines, doctor visits, lab tests, and hospital bills.

The system goes beyond simple tracking by analyzing spending data using Pandas, identifying patterns, and highlighting areas of high expenditure. Based on this analysis, it provides actionable insights such as cost-saving suggestions and personalized insurance recommendations.

Additionally, the application includes a rule-based conversational chatbot that allows users to interact with the system using natural language queries, making the experience more intuitive and user-friendly.

🎯 Objectives

Track and organize healthcare expenses
Analyze spending behavior
Provide intelligent financial insights
Suggest suitable insurance plans
Enable natural language interaction via chatbot
